# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project focuses on predicting house prices using Machine Learning techniques. 
The model is trained on housing dataset features such as area, number of bedrooms, bathrooms, location factors, etc., to estimate accurate property prices.

The objective of this project is to understand data preprocessing, feature selection, model building, and performance evaluation in a real-world regression problem.

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
